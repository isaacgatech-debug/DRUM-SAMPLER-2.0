QUICK INSTALLATION
------------------
1. Copy "Drum Sampler 2.component" to:
   /Library/Audio/Plug-Ins/Components/

2. Copy "Drum Sampler 2.vst3" to:
   /Library/Audio/Plug-Ins/VST3/

3. Copy "Drum Sampler 2.app" to:
   /Applications/

4. Restart your DAW

FIRST LAUNCH
------------
1. Open the plugin in your DAW or launch standalone
2. Click "LOAD SAMPLES" to select drum samples
3. Start playing!

IMPORTANT: macOS Security
-------------------------
If you see "cannot be opened because it is from an unidentified developer":
→ Go to System Preferences → Security & Privacy → Click "Open Anyway"
