# Drum Tech V1 Known Limitations

- Filename heuristics drive mapping; no manifest support in v1.
- Grid trigger editor is not yet a full MIDI piano-roll workflow.
- Trigger engine remains compiled in codebase but removed from normal UX flow.
- Some sample packs may need naming normalization to hit articulation/style filters.
- Font deprecation warnings from JUCE defaults remain and do not block runtime.
